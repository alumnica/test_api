# Test API

## Prerrequisitos
- Python 3.6
- virtualenv
- python-pip

## Configuración de entorno virtual (opcional)
```
$ virtualenv .venv
$ source .venv/bin/activate
```

## Instalación
### Dependencias
```
$ pip install -r requirements.txt

```


##crear DB
```
$ python manage.py makemigrations
$ python manage.py migrate
```

## Ejucutar
$ python manage.py runserver
